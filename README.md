# MacVault Optimizer

**A premium, safe, native macOS storage cleaner and optimizer.**

Reclaim gigabytes by intelligently removing temporary files, caches, hidden junk, duplicates, and large unused files — with full previews, undo, and enterprise-grade safety.

Built with **Swift + SwiftUI** for a truly Apple-native experience.

## Status
**Phase 0: Planning & Skeleton** (June 2026)

## Features (Planned)
- Visual disk usage (treemap/sunburst)
- Safe one-click + deep cleanup modules
- AI-assisted duplicate detection (local)
- Full preview + undo via snapshots
- Scheduled automation
- macOS Sequoia+ native

## Tech Stack
- **Language**: Swift 6
- **UI**: SwiftUI + AppKit where needed
- **Target**: macOS 14+
- **Architecture**: MVVM + Clean Architecture

## Quick Start (Development)
